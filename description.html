Project Description

Executive Summary: The Auto-Spotter is a state-of-the-art system designed to improve gym safety through automated spotting technology. It integrates advanced sensors, machine health diagnostics, and a winch and pulley mechanism to assist users during workouts.

Constraints and Requirements: The system must be reliable, easy to install, and compatible with a range of gym equipment. It should also feature user-friendly controls and robust safety mechanisms.

Design Approach: The design leverages accelerometers to detect motion, a robust winch and pulley system for load-bearing, and a machine health subsystem for real-time diagnostics.
